# Demo Source Document

RAG Foundation is a mock-mode document assistant for reviewing the product flow
without external SaaS credentials.

The demo flow shows four applied-AI engineering concerns:

- Authenticated users create isolated document stores.
- Uploaded documents are ingested through the backend and worker pipeline.
- Chat answers stream back through the API with source citations.
- Cost and budget visibility stays visible while users ask questions.

In demo mode, Gemini calls are replaced by deterministic mock responses. This
lets reviewers validate the application flow without API keys, billing setup, or
model latency.
